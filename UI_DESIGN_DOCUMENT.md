# Alpha 积分管理系统界面设计文档

## 1. 概述

本文档详细描述了 Alpha 积分管理系统桌面应用程序的用户界面设计，包括布局、组件、样式和交互方式。开发者可以根据此文档使用任何 GUI 框架重新实现该界面。

## 2. 界面布局

应用程序采用三区域布局设计：

1. **顶部区域**：无标题栏（已移除）
2. **中部区域**：水平排列的两个卡片面板
   - 左侧面板：日历视图
   - 右侧面板：当日数据
3. **底部区域**：全局统计数据横幅

在小屏幕设备上，中部区域会自动调整为垂直排列。

### 2.1 主窗口规格

- 固定尺寸：1200px × 900px
- 标题：Alpha 积分管理系统
- 背景：白色 (#FFFFFF)

## 3. 组件设计

### 3.1 卡片组件 (Card)

所有主要面板都采用卡片设计：

- 背景色：半透明白色 (rgba(255, 255, 255, 0.7))
- 边框：1px 灰色 (#94A3B8，透明度 64%)
- 圆角：14px
- 内边距：14px
- 阴影效果：
  - 模糊度：20px
  - 偏移：水平 0px，垂直 6px
  - 颜色：rgba(2, 8, 23, 0.4)

### 3.2 日历面板 (Calendar Card)

#### 3.2.1 月份导航栏

- 包含两个按钮：
  - "上一月" 按钮
  - "下一月" 按钮
- 月份标题标签：格式为 "YYYY 年 MM 月"
- 按钮样式：
  - 内边距：8px 12px
  - 圆角：10px
  - 边框：1px 灰色 (#94A3B8，透明度 64%)
  - 背景色：白色 (#FFFFFF)
  - 悬停背景色：浅灰 (#F9FAFB)

#### 3.2.2 星期标题行

- 7个标签，分别显示：一、二、三、四、五、六、日
- 文本对齐：居中
- 字体颜色：#64748B
- 字体大小：14px

#### 3.2.3 日期网格

- 6行7列的表格布局
- 单元格尺寸：正方形（宽高相等）
- 无表头显示
- 无网格线
- 单元格内边距：6px

##### 3.2.4 日期单元格状态

1. **普通日期**：
   - 文本颜色：黑色 (#000000)
   - 背景色：透明

2. **今日**：
   - 左上角显示绿色圆点标记
   - 圆点颜色：#50CC09
   - 圆点尺寸：6px × 6px

3. **有数据的日期**：
   - 背景色：蓝色渐变（从左上到右下）
     - 有效日期：从 #60A5FA 到 #3B82F6
     - 无效日期：从 #47559B 到 #475555
   - 文本颜色：白色

4. **选中日期**：
   - 蓝色边框：#3B82F6，2px 宽度
   - 有数据时叠加边框
   - 无数据时显示半透明蓝色背景

5. **修改过的日期**：
   - 右下角显示状态点
   - 修改状态：淡粉色 (#FBCFE8)
   - 未修改状态：淡绿色 (#A7F3D0)
   - 状态点尺寸：6px × 6px

### 3.3 当日数据面板 (Today Card)

#### 3.3.1 标题栏

- 标题文本："当日数据："
- 日期显示：格式为 "YYYY-MM-DD"
- 设置按钮：
  - 图标：齿轮图标 (setting_32px.ico)
  - 尺寸：24px × 24px
  - 图标尺寸：20px × 20px

#### 3.3.2 统计数据网格

- 2行4列网格布局
- 单元格间距：8px
- 每个单元格都是一个小卡片

##### 3.3.3 统计卡片 (Stat Card)

- 背景色：半透明白色 (rgba(255, 255, 255, 0.7))
- 边框：1px 灰色 (#94A3B8，透明度 64%)
- 圆角：12px
- 内边距：10px
- 阴影效果：
  - 模糊度：14px
  - 偏移：水平 0px，垂直 6px
  - 颜色：rgba(2, 8, 23, 0.4)

###### 3.3.4 统计卡片内容

1. 标签文本：
   - 字体颜色：#64748B
   - 字体大小：12px
   - 对齐方式：居中

2. 数值文本：
   - 字体颜色：#0F172A
   - 字体大小：22px
   - 字体粗细：600
   - 对齐方式：居中

###### 3.3.5 8个统计数据项

1. 初始资金
2. 完成资金
3. 磨损金额
4. 代币名称
5. 余额积分
6. 交易积分
7. 当日积分
8. 利润

### 3.4 全局统计横幅 (Global Summary)

- 位于窗口底部
- 包含5个统计项卡片：
  1. 总磨损
  2. 净利润
  3. 有效积分
  4. 总交易量
  5. 磨损率
- 每个卡片：
  - 内边距：12px
  - 圆角：14px
  - 阴影效果：
    - 模糊度：14px
    - 偏移：水平 0px，垂直 6px
    - 颜色：rgba(2, 8, 23, 0.4)
- 标题文本：
  - 字体大小：16px
  - 字体粗细：粗体
  - 对齐方式：居中
- 数值文本：
  - 字体大小：22px
  - 字体粗细：粗体
  - 对齐方式：居中

## 4. 弹窗设计

### 4.1 输入对话框 (Input Dialog)

#### 4.1.1 基本样式

- 窗口标题：根据操作类型变化
- 模态窗口
- 圆角：14px
- 边框：1px 灰色 (#94A3B8，透明度 64%)
- 背景色：半透明白色 (rgba(255, 255, 255, 0.8))

#### 4.1.2 选项卡区域

- 3个选项卡按钮水平排列：
  1. 计算 (图标: 2.ico)
  2. 使用 (图标: 7.ico)
  3. 修改 (图标: 8.ico)
- 按钮样式：
  - 可选中状态
  - 无间距排列
  - 默认背景色：白色
  - 选中背景色：蓝色 (#3B82F6)
  - 选中文本色：白色

#### 4.1.3 表单区域

##### 4.1.4 计算/修改表单

- 标题："输入当日数据" 或 "修改当日数据"
- 字段：
  1. 初始资金 (文本框，占位符："例如 1000")
  2. 完成资金 (文本框，占位符："例如 1200")
  3. 交易量 (文本框，占位符："例如 200")
  4. 磨损 (文本框，占位符："例如 5.000")
  5. 余额积分 (文本框，占位符："例如 3")
  6. 交易积分 (文本框，占位符："例如 4")
  7. 利润 (文本框，占位符："例如 180.000")
  8. 时间 (日期时间选择器，格式："yyyy-MM-dd HH:mm:ss")

##### 4.1.5 使用表单

- 标题："使用积分与代币"
- 字段：
  1. 代币名称 (文本框，占位符："例如 BTC")
  2. 代币数量 (文本框，占位符："例如 0.05")
  3. 代币价值 (文本框，占位符："例如 68000")
  4. 使用积分 (文本框，占位符："例如 10")
  5. 时间 (日期时间选择器，格式："yyyy-MM-dd HH:mm:ss")

##### 4.1.6 修改特有组件

- 选择条目下拉框
- 代币名称字段 (在修改选项卡中)

#### 4.1.7 操作按钮

- 3个主要操作按钮：
  1. 计算保存 (仅在计算选项卡显示)
  2. 保存使用 (仅在使用选项卡显示)
  3. 保存修改 (仅在修改选项卡显示)
- 按钮样式：
  - 背景色：蓝色 (#3B82F6)
  - 文本颜色：白色
  - 圆角：4px

## 6. 字体规范

- 主要字体族：'Segoe UI', Roboto, Arial
- 标题字体大小：16px (月份标题) / 18px (面板标题)
- 统计标签字体：12px
- 统计值字体：22px
- 按钮字体：默认大小

## 7. 交互设计

### 7.1 日历交互

- 单击日期：选中日期并更新右侧当日数据面板
- 双击日期：
  - 当日：打开输入对话框
  - 非当日：提示仅限当日操作

### 7.2 数据面板交互

- 双击统计数据卡片：打开编辑对话框

### 7.3 导航交互

- 点击"上一月"/"下一月"：切换月份显示

## 8. 响应式设计

- 大屏幕 (>1040px)：水平布局
- 小屏幕 (<960px)：垂直布局
